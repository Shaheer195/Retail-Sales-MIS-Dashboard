Retail Sales MIS Dashboard – UAE Electronics Store
🔹 Project Overview

This project focuses on building an Executive Management Information System (MIS) dashboard for a UAE-based electronics retail store. The dashboard helps management track sales performance, product categories, customer behavior, and regional trends to support data-driven decision-making.

🎯 Objectives

Analyze total revenue and sales performance by product category

Identify regional sales trends across UAE cities

Monitor return rates and customer behavior

Provide executive-level KPIs and business insights

🛠 Tools & Technologies

Excel – Data cleaning and preparation

SQL – Data aggregation and KPI calculation

Power BI – Dashboard creation and visualization

📁 Dataset

Open-source retail sales dataset (Kaggle / simulated UAE e-commerce data) including fields:

OrderID

ProductID

CustomerID

Product Category

Region (Dubai, Abu Dhabi, Sharjah)

SaleDate

Quantity

Revenue

Returns

🔄 Data Preparation

Removed null values and special characters in Excel

Created lookup tables for product categories and regions

Exported cleaned data to CSV

Used SQL queries to calculate:

Total Revenue & Units Sold by Category

Monthly Sales Trends

Return Rate by Category

Executive KPIs (MIS Metrics)

📈 Dashboard Pages

Home Page – Navigation & overview

Retail Sales Dashboard – Revenue, orders, trends, regional map

Product & Customer Analysis – Units sold, new vs repeat customers, gender-based revenue

Executive Performance Insights – KPIs and strategic summary

💡 Business Insights

Electronics contributes the highest share of revenue

Clothing category shows higher return rates

Dubai is the top-performing sales region

Sales peaks observed during seasonal periods

Opportunity to reduce revenue leakage through return management
